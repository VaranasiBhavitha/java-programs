# Java Access Control Program

## 📌 Description
This Java program demonstrates the use of **Access Modifiers (Access Control)** in Java. It explains how `public`, `private`, `protected`, and `default` access modifiers control the accessibility of classes, methods, and variables.

## 🎯 Concepts Covered
- Public Access Modifier
- Private Access Modifier
- Protected Access Modifier
- Default (Package-Private) Access Modifier

## 💻 Language
- Java

## ▶️ How to Run

1. Compile the program:
```bash
javac AccessControl.java
```

2. Run the program:
```bash
java AccessControl
```

## 📚 Output
The program demonstrates the accessibility of different access modifiers in Java.
