# Java Inheritance

This repository contains Java programs demonstrating the concept of **Inheritance**, one of the four fundamental principles of Object-Oriented Programming (OOP). These examples help beginners understand how classes can inherit properties and methods from other classes.

## 📌 Topics Covered

- Single Inheritance
- Multilevel Inheritance
- Hierarchical Inheritance
- Method Overriding
- super Keyword
- Constructor Inheritance
- Accessing Parent Class Members

## 🛠️ Technologies Used

- Java
- JDK 17 (or your installed version)
- IntelliJ IDEA / Eclipse / VS Code

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/VaranasiBhavitha/Inheritance.git
```

2. Open the project in your preferred Java IDE.

3. Compile and run any Java file.

Example:

```bash
javac SingleInheritance.java
java SingleInheritance
```
