# Arrays in Java

This folder contains Java programs that demonstrate various Array concepts. Arrays are used to store multiple values of the same data type in a single variable and are one of the fundamental concepts in Java programming.

## 📚 Topics Covered

- Array Declaration and Initialization
- Array Traversal
- Taking User Input in Arrays
- Finding Maximum and Minimum Elements
- Sum and Average of Array Elements
- Searching Elements in an Array
- Sorting Arrays
- Reversing Arrays
- Multi-Dimensional Arrays

## 📖 What is an Array?

An array is a data structure that stores multiple values of the same data type in a single variable.

Example:

```java
int[] numbers = {10, 20, 30, 40, 50};

for(int i = 0; i < numbers.length; i++) {
    System.out.println(numbers[i]);
}
```
